# Discord Bot (디스코드 봇) - BeTa

## 📘 개요

베타(Beta)는 여러분에게 **편하고 재미있는 기능들을 한곳에 모아** 도와드리는 Discord 봇이에요!  
TRPG 시트를 불러오거나, 메이플스토리 유저 정보를 찾아드리고, 버스가 언제 오는지도 알려드릴 수 있어요.  
또한 날씨를 확인해드리거나 치지직 생방송 여부도 알려드리며,  
로또 번호 생성·주사위 굴리기·가위바위보·롤 포지션 배정처럼 **즐길 거리도 가득** 준비돼 있어요.  

언제든 편하게 불러주세요! **BeTa**가 여러분을 도와드릴게요 😄

# 🎮 Discord Bot Setup Guide

이 문서는 Discord Bot을 실행하기 위한 기본 설치 및 환경 구성 방법을 정리합니다.


## 📌 1. Discord Developer Portal에서 봇 토큰 발급

1. [디스코드 개발자 링크](https://discord.com/developers/applications)에 접속합니다.
2. **New Application**을 생성합니다.
3. 좌측 메뉴에서 **Bot** 탭을 선택합니다.
4. **Reset Token** 또는 **Copy Token**을 눌러 봇 토큰을 발급받습니다.
5. 발급한 토큰은 외부에 공개되면 안 되며, `.env.example`을 참고하여 `.env` 파일에 저장합니다.


## 📌 2. Python 가상환경(venv) 설정

### ▶️ 가상환경 생성
```bash
python -m venv venv
```
### ▶️ 가상환경 활성화
```bash
venv\Scripts\activate
```
### Windows
```bash
venv\Scripts\activate
```
### macOS / Linux
```bash
source venv/bin/activate
```

## 📌 3. requirements.txt 설치
터미널 실행 후, 아래 코드를 입력합니다.  
프로젝트에 필요한 패키지는 requirements.txt에 포함되어 있습니다.
```bash
pip install -r requirements.txt
```

## 📌 4. 환경 변수 설정 (.env)
아래 형식으로 .env 파일을 생성하여 API 키나 토큰을 저장합니다.
```bash
DISCORD_BOT_TOKEN=your_token_here
```

## 📌 5. 디스코드 봇 실행
터미널 실행 후, 아래 코드로 봇 실행이 가능합니다.
```bash
python beta.py
```
또는
```bash
python3 beta.py
```

# 🎮 베타(Beta) 기능 명령어 모음

## 🚀 기본 기능
| 명령어 | 설명 |
|--------|-------|
| `/도움` | 베타가 학습한 기능 목록을 안내해드려요. |
| `/ping` | pong을 출력해 정상 동작 여부를 확인해요. |
| `/안녕` | 인사를 건네드려요. |
| `/임베드` | Discord 임베드 사용법 예시를 보여드려요. |
| *(이모티콘 사용)* | 이모티콘을 보내면 확대해서 보여드려요. |

---

## 🎲 랜덤 & 게임 기능
| 명령어 | 설명 |
|--------|-------|
| `/roll` | 1d100 다이스를 굴려요. |
| `/로또 (숫자)` | 입력한 횟수만큼 로또 번호를 생성해드려요. |
| `/가위바위보 (가위/바위/보)` | 베타와 가위바위보를 할 수 있어요. |

---

## 🌐 정보 조회 기능
| 명령어 | 설명 |
|--------|-------|
| `/시트` | TRPG 시트(구글 스프레드시트 링크)를 불러드려요. |
| `/날씨 (도시이름)` | 입력한 도시의 현재 날씨를 알려드려요. |
| `/치지직 (채널_이름)` | 한국 스트리밍 사이트 **치지직** 채널의 생방송 여부를 알려드려요. |

---

## 🧩 메이플스토리 기능

### 🔍 기본 조회
| 명령어 | 설명 |
|--------|-------|
| `/메이플` | 메이플 월드의 유저 정보를 조회해요. |
| `/메이플 (닉네임)` | 해당 유저의 기본 정보를 찾아드려요. |
| `/메이플링크 (닉네임)` | 입력한 유저의 링크 정보를 알려드려요. |
| `/메이플어빌 (닉네임)` | 유저의 어빌리티 정보를 조회해요. |

### 👥 보스 파티 관리
| 명령어 | 설명 |
|--------|-------|
| `/메이플_파티_등록 (보스_이름) (파티원_이름)` | 해당 보스 파티에 파티원을 등록해요. |
| `/메이플_파티_조회 (보스_이름)` | 보스의 파티 정보를 조회해요. |
| `/메이플_파티_삭제 (보스_이름) (파티원_이름)` | 해당 보스 파티에서 파티원을 삭제해요. |
